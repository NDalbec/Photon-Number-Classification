bayesianGaussianMixture#

class AutoencoderAPI.utils.clustering.bayesianGaussianMixture.bayesian_gaussianMixture(X_low, cluster_max, flip=False)[source]#

Use Bayesian Gaussian mixture to separate the latent space into regions of high density associated with photon events.

Parameters:

  • X_low (numpy.array) – Array containing all the samples in their low-dimensional representation.

  • cluster_max (float) – Maximum number of clusters to consider in the Bayesian Gaussian Mixture.

  • flip (bool) – If True flips the latent space. This can be used to re-order the labels (right to left).

Return type:

None
